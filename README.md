# hubot-bijin

A hubot script that does the things

See [`src/bijin.coffee`](src/bijin.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-bijin --save`

Then add **hubot-bijin** to your `external-scripts.json`:

```json
[
  "hubot-bijin"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
